# sales-performance-dashboard
An interactive dashboard built in Microsoft Excel to analyze key sales metrics such as total sales, profit, and product performance. It uses pivot tables, charts, and slicers to provide clear insights for better business decision making.
This project presents an interactive Sales Performance Dashboard built using Microsoft Excel to analyze and visualize key business metrics. The dashboard is designed to support data-driven decision-making by providing clear insights into sales trends, profitability, and product performance.

 Key Features:
KPI tracking (Total Sales, Profit, Profit Margin, Units Sold)
Monthly and yearly sales trend analysis
Product/category-wise performance comparison
Regional sales insights
Interactive filters and slicers for dynamic reporting
Visual elements including charts, icons, and conditional formatting

 Tools & Techniques Used:
Microsoft Excel (Pivot Tables, Pivot Charts)
Data Cleaning & Transformation
Dashboard Design Principles
Conditional Formatting
Slicers for interactivity

 Purpose:

The goal of this dashboard is to transform raw sales data into meaningful insights that help businesses monitor performance, identify trends, and make strategic decisions.

 Dataset:

Sample sales dataset used for analysis (can be customized for real world business use).
